#  🧾 **Retailer Sales Analysis Dashboard**

---

### **📊 Project Overview**

This project analyses a dataset of **9,648 Coca-Cola retailer records** across multiple **regions and states**, aimed at evaluating key **sales performance indicators (KPIs)**. The interactive Excel dashboard provides clear insights into:

* **Total Sales**
* **Units Sold**
* **Average Price per Unit**
* **Total Operating Profit**
* **Operating Profit Margin**

It further explores **sales performance by beverage brand** between two years and their **variance**, **sales by retailer** across two years and variance, and presents **sales and operating profit margin by quarters**.

### **🧪 Methodology**

* **Dataset:** 9,648 retailer profiles containing 12 fields:
  *Retailer, Retailer ID, Invoice Date, Region, State, City, Beverage Brand, Price per Unit, Units Sold, Total Sales, Operating Profit, Operating Margin*

* **Primary Tool:** Microsoft Excel

  * Pivot Tables
  * Dynamic Formulas and Conditional Formats
  * Pivot Charts
  * Slicers

* **Process:**

  1. Data preparation for accuracy and consistency.
  2. KPI computation for sales and profitability metrics.
  3. Dashboard development using pivot tables, charts, formula and conditional formatting.
  4. Integration of slicers for interactive filtering by region and year.

---

### **🧭 How to Use Dashboard**

1. Download `Retailer Project.xlsx` and open in Excel (Office 365 or Excel 2019+ recommended).
2. Use slicers to filter by **year (Invoice Date)** and **region**.
3. Explore KPI visuals to analyse sales trends and profitability.
4. Compare year-on-year performance and brand contributions interactively.
